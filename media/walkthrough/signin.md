# Sign in to YouTrack

Run the **YouTrack: Sign In** command from the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).

You'll be prompted for:

1. Your YouTrack Cloud **base URL** (e.g. `https://<org>.youtrack.cloud/`).
2. A **permanent token**, created at **YouTrack → avatar → Profile → Account Security → New token**. Scope it to **YouTrack** (not read-only).

Once signed in, the YouTrack icon in the activity bar activates and the sidebar populates with your issues, notifications, and agile boards.
