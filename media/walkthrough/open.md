# Open your first issue

Expand **Assigned to me** in the sidebar, or run **YouTrack: Go To Issue** (`Ctrl+Alt+G` / `Cmd+Alt+G`) and enter an ID like `ABC-123`.

The issue detail panel opens with:

- Editable summary and description (double-click to edit, Markdown supported).
- Clickable pills on the side: State, Priority, Assignee, Tags, any custom field.
- Activity feed with comments, logged time, and field-change history.
- Drag-and-drop attachments onto the panel, or paste a screenshot into a comment.
