# Customize

A few settings are worth knowing about:

- `youtrack.defaultProject` — short name used when creating issues.
- `youtrack.branch.template` / `youtrack.branch.summaryMaxLength` / `youtrack.branch.separator` — branch-naming policy.
- `youtrack.commit.template` / `youtrack.commit.autoFill` — SCM input prefix (`off` / `empty-only` / `always`).

Useful keybindings:

| Action | Windows / Linux | macOS |
|---|---|---|
| Create issue | `Ctrl+Alt+N` | `Cmd+Alt+N` |
| Go to issue by ID | `Ctrl+Alt+G` | `Cmd+Alt+G` |
| Search issues | `Ctrl+Alt+Y` | `Cmd+Alt+Y` |
| Open board | `Ctrl+Alt+B` | `Cmd+Alt+B` |
| Peek issue under cursor | `Ctrl+Alt+P` | `Cmd+Alt+P` |

Templates: drop markdown files into `.youtrack/templates/` in your workspace — they show up in the Create Issue panel's Template dropdown.
